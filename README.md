# Clients_CRM
Система управления контактными данными клиентов - диплмная работа по завершении курса "Базовый Java Script" на образовательной платформе SkillBox. Вэб-приложение позваляет хранить список клиентов, изменять его, а также управлять данными клиентов.
#### Возможности:
* Просмотр списка клиентов в виде таблицы.
* Добавление и удаление клиентов.
* Редактирование контактных данных клиента.
* Вывод контактных данных клиента в виде отдельной карточки.
* Сортировка списка клиентов по различным полям таблицы.
* Валидация формы добавлени клиента и изменения его данных.
#### Установка:
* Загрузите приложение с помощью команды `git clone`
* запустите сервер, выполнив команду `node index` в папке crm-backend
* oткройте файл index.html в папке crm-frontend
